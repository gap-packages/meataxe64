  [1X3.1 [33X[0;0YFile-based Parallel Functions[133X[101X
  
  [33X[0;0YThe C Meataxe64 includes a number of parallel implementations of challenging
  computations.  These  take  their  inputs  and  return their results in disk
  files,  which  can  be  read and written using [2XMTX64_WriteMatrix[102X ([14X1.5-2[114X) and
  [2XMTX64_ReadMatrix[102X ([14X1.5-2[114X). They use a number of threads specified in the file
  [10Xsrc/mtx64/tuning.h[110X.  Each  requires  a  [3Xtmpdir[103X parameter which should be the
  pathname of a directory suitable for temporary files.[133X
